#Accessibility-First Responsive Website

A fully responsive and accessibility-focused website built with HTML, CSS, and JavaScript.
This project follows WCAG 2.1 guidelines and includes semantic HTML, ARIA labels, keyboard navigation, skip links, and high-contrast mode
